# AngularJs + Gulp

Build project angularJS with GULP.

### Installing

Install gulp cli

```
npm install gulp-cli -g
```

Install dependecies
```
npm install
```

### Running

Running gulp and BrowserSnc

```
gulp
```

Running server static

```
node server.js
```
